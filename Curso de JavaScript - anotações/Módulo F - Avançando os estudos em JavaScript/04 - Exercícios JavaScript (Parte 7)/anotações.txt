Restrições da aplicação de teste que devem ser seguidas:
=========================================================

- Não é aceito valor vazio.
- Não são aceitos números que não estejam entre 1 e 100.
- Não são aceitos números repetidos.
- Ao clicar no botão Finalizar, um relatório deve ser exibido para o usuário (como no print da tela na pasta imagens).
- Ao clicar no botão Finalizar, se não houver nenhum valor inserido pelo usuário, a mensagem "Adicione valores antes de finalizar" deve ser exibida.
- Ao adicionar um novo valor, o relatório para usuário deve desaparecer da tela.
